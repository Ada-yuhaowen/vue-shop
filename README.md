# vue-shop

#### 介绍

摸鱼六佬的前端项目，学习构建电商系统前端网站

项目结构
```
$ tree .
vue-shop
├─code  项目代码
├─docs  文档、学习资料、任务计划等
└─img   需要的图片
```
